Descripcion del proyecto


La aplicación permite a los usuarios ingresar sus datos personales mediante un formulario, realizar validaciones en tiempo real y visualizar los aspirantes registrados en una tabla dinámica. Todo el manejo de la información se realiza en memoria mediante estructuras tipo lista.



La aplicación también tiene un simulación de CDT que permite a los usuarios ingresar un monto de inversión y seleccionar el tipo de certificado (mensual, trimestral, semestral o anual). A partir de estos datos, el sistema calcula automáticamente la tasa de interés, el plazo en días, la ganancia generada, el impuesto aplicado y el valor final de la inversión.
